# My Simple Website

This is a simple website project that demonstrates the use of HTML, CSS, and JavaScript to create a basic web page.

## Project Structure

```
my-simple-website
├── src
│   ├── index.html        # Main HTML document for the website
│   ├── css
│   │   └── styles.css    # Styles for the website
│   ├── js
│   │   └── main.js       # JavaScript code for interactivity
│   └── fonts             # Directory for custom font files
├── .gitignore            # Files and directories to be ignored by Git
└── README.md             # Documentation for the project
```

## Setup

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd my-simple-website
   ```

3. Open the `src/index.html` file in your web browser to view the website.

## Usage

- The `index.html` file contains the structure of the webpage.
- The `styles.css` file defines the visual aspects of the webpage.
- The `main.js` file adds interactivity to the website.
